AffAuto Video - Privacy Policy

Last updated: May 31, 2026

This Privacy Policy explains how AffAuto Video handles user information when using the software.

1. Information We Collect

AffAuto Video may use information provided by the user, such as API keys, video files, captions, affiliate links, and platform credentials, only for the purpose of creating, processing, and uploading content as requested by the user.

2. TikTok Data

When users connect their TikTok account, the software may request basic profile access and video upload permission. The software uses these permissions only to authenticate the user and upload videos to TikTok as drafts.

3. Data Storage

User settings and API keys may be stored locally on the user's computer. The software does not sell user data to third parties.

4. Data Sharing

AffAuto Video does not share personal data with advertisers or third parties. Data may only be sent to platform APIs such as TikTok, Google, Facebook, or Gemini when the user performs actions that require those services.

5. User Control

Users can revoke platform permissions from their TikTok, Google, or Facebook account settings. Users can also remove local configuration files from their computer.
